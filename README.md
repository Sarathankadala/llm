# AI Legal Document Simplifier

A premium web application that converts complex legal text into plain English for educational purposes.

## Features

- **Legal Jargon Replacement**: Automatically converts 40+ legal terms into everyday language
- **Key Point Extraction**: Identifies obligations, restrictions, rights, penalties, and timelines
- **Risk Assessment**: Automatically evaluates High/Medium/Low risk levels
- **Premium UI**: Modern dark mode with glassmorphism and smooth animations
- **Copy to Clipboard**: Easy copying of all output sections
- **Sample Documents**: Pre-loaded examples for testing

## How to Use

1. Open the application in your web browser
2. Paste legal text into the input area (or click "Load Sample")
3. Click "Simplify Document" or press Ctrl+Enter
4. Review the simplified explanation, key points, and risk assessment
5. Use copy buttons to save any section

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Design**: Custom CSS with dark mode and gradients
- **Fonts**: Google Fonts (Inter)

## Deployment

This is a static web application that can be deployed to any hosting service:
- Vercel
- Netlify
- GitHub Pages
- Any static file server
